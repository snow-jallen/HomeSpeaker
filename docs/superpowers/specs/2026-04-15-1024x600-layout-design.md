# 1024×600 Display Layout Optimization

**Date:** 2026-04-15  
**Status:** Approved (Option C selected from mockups)

## Summary

Optimize the HomeSpeaker UI for a 1024×600 touchscreen. Four areas of change: play controls, nav sidebar, blood sugar card, and temperature monitor layout.

---

## 1. Play Controls (`PlayControls.razor`)

Remove the **Shuffle** and **Repeat** buttons.

**After:** Play · Stop · Skip (row 1) · Sleep timer · Delete (row 2)

---

## 2. Nav Sidebar (`NavMenu.razor` + `MainLayout.razor`)

**Link order:** Music · Queue · Streams · Playlists · YouTube · Anchors

**Home link:** Hidden on the home page (`/`), visible on all other pages.

**Sidebar width:** Reduced from 275px → 210px (desktop breakpoint).

---

## 3. Blood Sugar Card (`BloodSugarMonitor.razor`)

Replace the 3-column (value / trend / time) card layout with a compact horizontal card:

```
┌─────────────────────────────────────────────┐
│ 💉 Blood Sugar                    🔗 NightScout │
│  170   →   Stable                             │
│             4 min ago                         │
└─────────────────────────────────────────────┘
```

- **Value** (e.g. `170`): large (~2rem), colored by status
- **Arrow** (trend direction icon): same large size, colored
- **Direction** (e.g. `Stable`): medium size, colored
- **Time** (e.g. `4 min ago`): small, muted
- **NightScout link icon** (`🔗`) in card header, right-aligned — links to the NightScout page (`/nightscout`)

---

## 4. Temperature Monitor (`TemperatureMonitor.razor`)

Replace the 6-column uniform grid with a **two-panel layout**:

```
┌──────────────────────────────┬──────────────────┐
│  ☁ 62°   ♀ 71°               │  ☀ 81°           │
│  Outside  Girl's Room        │  Today's High    │
│                               │                  │
│  ♂ 70°   🌿 74°               │  🌙 55°          │
│  Boy's Room  Greenhouse      │  Tonight's Low   │
└──────────────────────────────┴──────────────────┘
```

- Left panel (`3fr`): 2×2 grid of the four sensor readings
- Right panel (`2fr`): forecast readings stacked vertically (Today's High / Tonight's Low, or Tomorrow's High)
- Each cell: **icon inline before value** (not above), label below
- Numbers larger than before (~1.8–2rem)
- A subtle vertical divider between the two panels

---

## 5. Home Page Layout (`Index.razor`)

- Blood sugar card: **full width**, directly below the Now Playing card
- Temperature monitor: **full width**, below blood sugar
- Remove the `col-md-6` side-by-side split between the two health widgets
